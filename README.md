# Hospital Management System

## Overview
This repository hosts a comprehensive Hospital Management System designed to streamline operations within healthcare facilities. It consists of two main components: a frontend built with Angular and a backend developed using Spring Boot. The system utilizes SQL databases managed through Hibernate for efficient data storage and retrieval.

## Features

### Frontend (Angular)
- User-friendly interface for managing patient records, appointments, and staff management.
- Responsive design ensuring compatibility across devices.
- Intuitive navigation and data visualization.

### Backend (Spring Boot)
- RESTful APIs facilitating seamless communication between frontend and backend.
- Business logic handling patient management, appointment scheduling, and administrative tasks.
- Secure endpoints with authentication and authorization mechanisms.

### Database
- SQL database integrated with Hibernate ORM for efficient data handling.
- Schema designed to support hospital-specific data structures such as patient records, medical history, and staff profiles.
